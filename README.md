# Myanmar cyclones Anticipatory Action trigger
In this repo:
- Analysis of historical observational data and CERF allocations used to have a cyclones overview and set-up a trigger
- Analysis of historical forecasts data to test the trigger 
- Workflow running daily on new forecasts data

# Data Sources used
Windspeed: ECMWF data
Rainfall: CHIRPS-GEFS (forecast) and IMERGE (observations)

# How to get started
`pip install -r requirements.txt`
